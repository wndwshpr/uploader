File Requirements

Required Columns

  • name - Product name (required)
  • price - Product price in numbers (required, e.g., 9.99)
  • stock - Available quantity (required, whole number)

Optional Columns

  • category - Product category (e.g., Electronics, Clothing)
  • barcode - Product barcode/SKU/UPC
  • description - Product description

Important Notes

  • Column names are case-insensitive (Name, name, NAME all work)
  • Alternative column names accepted: quantity/stock_quantity for stock, sku/upc for barcode
  • Invalid rows will be skipped with error details
  • Maximum 10,000 products per upload
  • Supported formats: CSV, XLSX, XLS
